WhatsApp Bot - Classic Nehy

A powerful WhatsApp bot built using Node.js and Baileys, designed to automate chats, respond to commands, and enhance user interaction.

💻 Features

- Auto-reply to messages
- Media support (images, audio, stickers)
- Anti-delete message system
- Auto-view and like statuses
- Fun commands
- AI integration
- autotyping
- autorecording
- And more...

🚀 Getting Started

Requirements

- Node.js v14+
- npm or yarn
- WhatsApp account
- Baileys (or your preferred library)

Installation

```bash
git clone https://github.com/yourusername/nehy-whatsapp-bot.git
cd nehy-whatsapp-bot
npm install
```

Start the Bot

```bash
node index.js
```

> Scan the QR code on first run to authenticate.

📁 Project Structure

- `index.js` - Main bot logic
- `session.json` / `auth/` - Session data
- `commands/` - Custom bot commands
- `README.md` - Project overview
